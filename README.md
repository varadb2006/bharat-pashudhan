# Bharat Pashudhan — Indigenous Breed Identification System

An AI-powered tool for identifying indigenous Indian cattle and buffalo breeds. 
Built for field-level agricultural workers across India.

## Architecture
- **AI Engine**: EfficientNet-B4 (PyTorch) — 66 breed classes
- **API**: FastAPI (Python)
- **Frontend**: React.js (coming soon)
- **Database**: MySQL (coming soon)

## Breeds Covered
- 49 indigenous cattle breeds
- 17 indigenous buffalo breeds
- Total: 66 breeds

## Model Performance
- Training images: 14,876 (deduplicated)
- Validation accuracy: 63.3%
- Architecture: EfficientNet-B4 + Deep Classifier Head
